Connect AI assistants to your ThinkPad via MCP

MCP Server

The MCP (Model Context Protocol) server lets AI assistants monitor and control your ThinkPad: fan speed, battery, CPU, temperatures, and more.

Checking...

Available Tools

get_fan_status Fan speed (RPM), level, status
set_fan_speed Set auto, full-speed, or level 0-7
get_cpu_temperature All thermal zone readings
get_battery_info Status, capacity, health, thresholds
set_battery_thresholds Set charge start/stop percentages
get_cpu_info Governor, frequency, turbo boost
get_memory_info RAM usage details
get_system_info Hostname, kernel, OS, CPU model

Setup Instructions

Start the MCP server above, then add it to your AI tool. Server URL: http://127.0.0.1:8765/sse

Run this command:

claude mcp add --transport sse thinkutils http://127.0.0.1:8765/sse

Or add to .mcp.json in your project:

{
  "mcpServers": {
    "thinkutils": {
      "type": "sse",
      "url": "http://127.0.0.1:8765/sse"
    }
  }
}

Add to ~/.config/Claude/claude_desktop_config.json: 

{
  "mcpServers": {
    "thinkutils": {
      "type": "sse",
      "url": "http://127.0.0.1:8765/sse"
    }
  }
}

ChatGPT Desktop supports MCP servers. Go to Settings > Features > MCP Servers, click Add, and enter: 

Server URL: http://127.0.0.1:8765/sse
Name: ThinkUtils

Requires ChatGPT Desktop with MCP support (Plus/Team/Enterprise).

In Cursor or Windsurf, go to Settings > MCP and add a new SSE server: 

Name: thinkutils
URL: http://127.0.0.1:8765/sse
Transport: SSE

For any MCP-compatible client, configure an SSE server with:

URL: http://127.0.0.1:8765/sse
Transport: SSE (Server-Sent Events)

Works with any tool supporting the Model Context Protocol including LM Studio, Continue.dev, and other MCP clients.
